# 房屋中介管理系统

一个基于 C 语言实现的课程设计项目，覆盖房源管理、租客注册与登录、看房预约、租约签订、审核与统计等中介业务流程。

## 主要功能

- 管理员管理中介、租客、房源、分类信息和系统维护
- 中介录入房源、处理预约、发起租约、维护业务数据
- 租客注册登录、筛选房源、预约看房、处理待签合同
- 密码使用带盐哈希存储，找回/管理员重置会发放临时密码
- 本地二进制数据持久化

## 构建

```bash
gcc -Wall -Wextra -Wpedantic -std=c11 main.c rental_system.c storage.c -o rental_system
```

## 运行

```bash
./rental_system
```
